Element.updateChild

dart
newWidget==null newWidget != null
child == null returns null. returns new [Element]
child != null 移除child,returns null 可复用就复用，returns child
不可复用，return new [Element]

各种条件下来其实就是 能更新就更新，否则移除旧的child，创建并返回新的Element。
